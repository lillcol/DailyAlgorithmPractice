# LeetCode 11 - 盛最多水的容器

## 题目描述

给定一个长度为 `n` 的整数数组 `height`，有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])`。

找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。返回容器能储存的最大水量。

**说明：** 你不能倾斜容器。

## 边界条件

- `n == height.length >= 2`
- 面积由较矮一侧的高度与底边宽度决定：`min(h[l], h[r]) * (r - l)`

## 多解法

### 一般解法

- 核心思路：枚举所有线对 `(i, j)`（`i < j`），计算面积取最大。
- 时间复杂度：`O(n^2)`
- 空间复杂度：`O(1)`

### 经典解法

- 核心思路：**双指针** 从数组两端向中间移动。当前面积为 `min(h[l], h[r]) * (r-l)`；每次移动 **较矮一侧** 的指针，因为宽度必然变小，只有可能换到更高的边才可能增大面积。
- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

### 最优解法

- 核心思路：与经典双指针相同，已是本题最优时间复杂度。

## 解题思路（以双指针为主）

### 思路概述

设左指针 `l`、右指针 `r`。面积受限于较矮边，若 `height[l] < height[r]`，则固定 `r` 左移任何位置宽度减小且高度仍不超过 `height[l]`，不可能更优，故令 `l++`；对称地若右边更矮则 `r--`。

### 关键步骤

1. `l = 0`，`r = n - 1`，`ans = 0`
2. 计算当前面积并更新 `ans`
3. 移动较矮一侧指针；相等时任移一侧即可

## 通俗白话讲解

短板决定水位。底边变窄时，只有把 **更短的那块板** 换掉，才有机会装更多水；动长板只会宽度变小、高度仍被短板卡住，不可能更好。

## 复杂度分析（对比）

- 一般解法：时间 `O(n^2)`，空间 `O(1)`
- 最优解法：时间 `O(n)`，空间 `O(1)`
