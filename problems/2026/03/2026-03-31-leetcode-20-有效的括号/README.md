# LeetCode 20 - 有效的括号

## 基本信息

- 日期：2026-03-31
- 难度：Easy
- 题型：栈
- 题目链接：[https://leetcode.com/problems/valid-parentheses/](https://leetcode.com/problems/valid-parentheses/)

## 题目描述

给定一个只包含字符 `(`、`)`、`[`、`]`、`{`、`}` 的字符串 `s`，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合
2. 左括号必须按正确顺序闭合

## 输入输出示例

### 示例 1

- 输入：`s = "()"`
- 输出：`true`

### 示例 2

- 输入：`s = "()[]{}"`
- 输出：`true`

### 示例 3

- 输入：`s = "(]"`
- 输出：`false`

## 边界条件

- 空字符串视为有效
- 只要出现“右括号先于匹配左括号”就立即无效
- 字符串长度为奇数时一定无效（括号无法两两配对）

## 多解法

### 一般解法

- 核心思路：反复在字符串中删除 `"()"`、`"[]"`、`"{}"` 直到不能删除；最后为空则有效。
- 时间复杂度：最坏 `O(n^2)`
- 空间复杂度：`O(n)`（字符串拷贝）

### 经典解法

- 核心思路：使用栈。遇到左括号入栈，遇到右括号检查是否与栈顶匹配，不匹配直接返回 `false`。
- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`（最坏全是左括号）

### 最优解法

- 核心思路：仍用栈，但遇到左括号时直接压入“期望的右括号”，遇到字符只需与栈顶比较即可。
- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 解题思路（以最优解法为主）

### 思路概述

遍历字符串：

- 如果是左括号，就把它对应的右括号压栈（例如遇到 `(` 压入 `)`）
- 如果是右括号，就必须与栈顶相同，否则无效
- 全部遍历结束后，栈必须为空才有效

### 关键步骤

1. 初始化空栈
2. 遇到 `(`、`[`、`{`，分别压入 `)`、`]`、`}`
3. 遇到右括号时，若栈空或不等于栈顶，返回 `false`
4. 最后检查栈是否为空

## 通俗白话讲解

把栈想成“待完成清单”。  
每遇到一个左括号，就记下“我后面期待哪个右括号”。  
当右括号出现时，必须先满足最近的一条期待（后进先出），否则顺序就错了。  
最后如果期待都被满足（栈空），说明整个字符串有效。

## 复杂度分析（对比）

- 一般解法：时间 `O(n^2)`，空间 `O(n)`
- 最优解法：时间 `O(n)`，空间 `O(n)`
