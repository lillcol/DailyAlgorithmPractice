# LeetCode 3 - 无重复字符的最长子串

## 基本信息

- 日期：2026-03-26
- 难度：Medium
- 题型：滑动窗口
- 题目链接：[https://leetcode.com/problems/longest-substring-without-repeating-characters/](https://leetcode.com/problems/longest-substring-without-repeating-characters/)

## 题目描述

给定一个字符串 `s`，请你找出其中不含有重复字符的最长子串的长度。

## 输入输出示例

### 示例 1

- 输入：`s = "abcabcbb"`
- 输出：`3`
- 解释：无重复字符的最长子串是 `"abc"`，长度为 3。

### 示例 2

- 输入：`s = "bbbbb"`
- 输出：`1`
- 解释：无重复字符的最长子串是 `"b"`，长度为 1。

### 示例 3

- 输入：`s = "pwwkew"`
- 输出：`3`
- 解释：无重复字符的最长子串是 `"wke"`，长度为 3。

## 边界条件

- `s` 可能是空字符串，此时结果为 `0`
- `s` 可能所有字符都相同，此时结果为 `1`（非空时）
- `s` 可能本身就没有重复字符，此时结果为 `s.length()`

## 多解法

### 一般解法

- 核心思路：枚举所有子串，并逐个判断是否包含重复字符。
- 时间复杂度：`O(n^3)`（枚举子串 + 判重）
- 空间复杂度：`O(k)`（判重时临时集合，`k` 为字符集大小）

### 经典解法

- 核心思路：滑动窗口 + 集合，右指针扩张，遇到重复时左指针逐步收缩直到无重复。
- 时间复杂度：`O(2n)`，可视为 `O(n)`
- 空间复杂度：`O(k)`

### 最优解法

- 核心思路：滑动窗口 + 哈希表记录字符最近下标，重复时左边界直接跳跃而非逐步移动。
- 时间复杂度：`O(n)`
- 空间复杂度：`O(k)`

## 解题思路

### 思路概述

使用滑动窗口维护一个“当前不重复子串”区间，用哈希表记录每个字符最近出现的位置。  
当遇到重复字符时，把窗口左边界直接跳到重复字符上次位置的后一位。

### 关键步骤

1. 用 `left` 表示窗口左边界，遍历指针 `right` 从左到右扫描字符串。
2. 每次拿到当前字符 `c`，如果它在窗口内出现过，就更新 `left`。
3. 每轮更新当前窗口长度 `right - left + 1`，并维护最大值。

## 通俗白话讲解

可以把窗口理解成一条“橡皮筋”，里面只能放不重复字符。  
右指针每次往右扩一格，尝试把新字符放进来。  
如果发现和窗口里某个字符撞车了，就把左指针快速跳到撞车字符上次位置的后面，  
这样窗口里又恢复成“全都不重复”的状态，然后继续往右扩。

## 复杂度分析（对比）

- 一般解法：时间 `O(n^3)`，空间 `O(k)`
- 最优解法：时间 `O(n)`，空间 `O(k)`
